NGX-Time-Since

Date transformation

The timeSince pipe transforms a date value into a text description of the time since that date value.

The pipe is useful for timestamp data such as comments, images, or updates.

@for(ex of examples; track ex) {
The input date of {{ ex | date:"MM/dd/yyyy 'at' HH:mm:ss a" }} becomes {{ ex | timeSince }}
}
